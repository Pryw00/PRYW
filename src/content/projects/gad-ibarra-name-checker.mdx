Herramienta web desarrollada para el GAD Municipal San Miguel de Ibarra que permite a ciudadanos y funcionarios verificar la disponibilidad de nombres por sector dentro del sistema catastral municipal.

La aplicación se integra con el sistema SIGM (Sistema de Información y Gestión Municipal) de Ibarra, proporcionando consultas en tiempo real y simplificando trámites que antes requerían atención presencial.
